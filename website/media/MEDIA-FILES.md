# Media Files — Required for Website

Place all image/GIF files in this folder (`website/media/`).
File names must match exactly (case-sensitive).

---

## App Icon

| File | Description |
|---|---|
| `app-icon.png` | The app's launcher icon. Export from Android Studio at 512×512 px. Used in navbar, footer, and browser tab favicon. |

---

## Hero Image

| File | Description |
|---|---|
| `hero-phone.png` | A phone mockup showing the keyboard in use. Recommended: Pixel phone frame with the keyboard visible on screen. Transparent or dark background. Min width 600 px. |

## Tips

- Recommended screenshot dimensions: 1080 × 2400 px (or any modern Android resolution)
- Save all images as PNG for screenshots; GIF for animations
- Keep total media folder size reasonable (< 20 MB) for fast page loads
- Optimise PNGs with tools like TinyPNG or pngquant before uploading
